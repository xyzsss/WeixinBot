###Install
`pip install -r requirements.txt`
The requirements file was generated by `pip freeze > requirements.txt`


###Run
python weixin.py

####Run background
For debian :
```
apt-get install tmux
tmux 
python weixin.py
Ctrl + b + d
```
To reattach the tmux session with `tmux a`
